Неизменяемые объекты бесплатно обеспечивают атомарность (раз­ дел 10.8). Их состояние никогда не изменяется, так что нет никакой возмож­ности получить временную несогласованность.

Основным недостатком неизменяемых классов является то, что они требуют отдельный объект для каждого уникального значения. Создание этих объектов может быть дорогостоящим, особенно если они большие.
